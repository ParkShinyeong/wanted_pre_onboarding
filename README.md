# wanted_pre_onboarding

원티드 프리온보딩 사전 과제입니다. Nest JS와 TypeORM을 사용하였습니다.

### DB Schema

![DB Schema](./image/wanted_preonboarding_3_db_schema.png)

**users** - 사용자 정보 테이블 <br />
**recruitments** - 채용 공고 테이블 <br />
**companies** - 회사 정보 테이블 <br />
**apply_histories** - 채용 공고 지원 현황 테이블 <br />
